Interaktive HTML-Lernumgebung

Quadratische Ergänzung als Flächenmodell

Diese Lernumgebung führt Schüler*innen Schritt für Schritt durch die Gleichung x² + 10x − 39 = 0. Die algebraischen Umformungen werden dabei eng mit einem Flächenmodell verknüpft – so wie auf der Tafel im Seminar.

Arbeitsauftrag – Ausarbeitung

Unterhalb der interaktiven Lernumgebung findest du eine ausformulierte Bearbeitung des Seminarauftrags: Initialprompt, iterative Optimierung, didaktische Begründung und Hinweise zur lokalen Ausführung.

Didaktischer Fokus

  1. Vom Term zur Fläche
  2. Umordnen statt „blind umformen“
  3. Fehlende Ecke bewusst ergänzen
  4. Rückübersetzung in Algebra
  5. Reflexion: Geometrie vs. algebraische Vollständigkeit

Wichtiger Hinweis

Das Flächenmodell arbeitet mit positiven Längen. Deshalb erscheint geometrisch vor allem die Lösung x = 3. Algebraisch gilt jedoch:

x = 3  oder  x = -13

Die zweite Lösung entsteht beim Lösen von (x+5)² = 64 durch x+5 = ±8. Sie ist im Flächenmodell als Seitenlänge nicht direkt interpretierbar.

1. Geeigneter Initialprompt für das LLM

Erstelle eine eigenständige HTML-Datei (HTML, CSS und JavaScript in einer Datei) für eine interaktive Lernumgebung zur quadratischen Ergänzung. Zielgruppe sind Schüler*innen der Sekundarstufe. Ausgangsgleichung ist x² + 10x - 39 = 0. Die Lernumgebung soll die Umformungen eng mit einem Flächenmodell verknüpfen: - Stelle x² als Quadrat dar. - Stelle 10x als zwei Rechtecke der Maße 5·x dar. - Ordne die Rechtecke so um, dass fast ein großes Quadrat entsteht. - Zeige, dass zum vollständigen Quadrat die Ecke 5·5 = 25 ergänzt werden muss. - Verbinde jeden visuellen Schritt mit der passenden algebraischen Umformung. - Zeige anschließend die Rückübersetzung in die Form (x+5)² = 64. - Löse die Gleichung schrittweise und erläutere, warum das Flächenmodell vor allem zur positiven Lösung x = 3 führt, während algebraisch auch x = -13 entsteht. Anforderungen: - Schritt-für-Schritt-Navigation mit Zurück/Weiter. - Interaktive Visualisierung mit SVG oder Canvas. - Leicht verständliche didaktische Sprache. - Kurze Kontrollfragen oder Eingaben für Lernende. - Farbliche Unterscheidung von x², Rechtecken und Ergänzungsfläche. - Responsives Layout. - Keine externen Bibliotheken. - Sauber kommentierter Code. Gib nur den vollständigen lauffähigen HTML-Code aus.

2. Iterativer Prompting-Prozess

  1. Erste Optimierung: „Überarbeite die Lernumgebung sprachlich. Formuliere alle Erklärungen schülernah, in kurzen Sätzen und mit klarer Handlungsorientierung. Jede Umformung soll mit einer Beobachtung am Flächenmodell verbunden werden.“
  2. Zweite Optimierung: „Verbessere die Visualisierung. Nutze SVG, beschrifte alle Seitenlängen deutlich mit x bzw. 5 und markiere die Ergänzungsfläche 25 farblich auffällig.“
  3. Dritte Optimierung: „Ergänze zu jedem Schritt eine kleine Lernfrage mit unmittelbarem Feedback. Die Fragen sollen das Ergänzen zum Quadrat sichern.“
  4. Vierte Optimierung: „Füge eine didaktische Reflexion ein: Warum zeigt das Flächenmodell vor allem die positive Lösung? Wie kommt algebraisch die negative Lösung zustande?“
  5. Fünfte Optimierung: „Strukturiere den Code sauber mit kommentierten Abschnitten und achte auf eine lokal lauffähige Ein-Datei-Lösung ohne externe Abhängigkeiten.“

3. Erklärung der Vorgehensweise

Ausgangspunkt: Die Gleichung x² + 10x − 39 = 0 wird zuerst zu x² + 10x = 39 umgeformt. So steht links die Fläche aus einem Quadrat x² und einem Rechteckinhalt 10x, rechts die Vergleichsfläche 39.

Geometrische Deutung: Das Rechteck 10x wird in zwei gleich große Rechtecke 5x zerlegt. Diese werden an zwei Seiten des Quadrats angelegt. Dadurch entsteht fast ein großes Quadrat, nur eine Ecke fehlt.

Quadratische Ergänzung: Die fehlende Ecke hat die Seitenlängen 5 und 5, also den Flächeninhalt 25. Deshalb ergänzt man auf beiden Seiten der Gleichung 25. So bleibt die Gleichung äquivalent.

Rückübersetzung: Links ist nun ein vollständiges Quadrat mit Seitenlänge x + 5. Deshalb gilt (x+5)² = 64.

Lösen: Algebraisch folgt x + 5 = ±8, also x = 3 oder x = -13. Im Flächenmodell ist jedoch nur x = 3 direkt sinnvoll, weil Seitenlängen dort positiv gedacht werden.

Didaktische Bewertung: Die Lernumgebung verbindet symbolische Umformung und anschauliches Handeln. Die Schüler*innen sehen nicht nur, dass man 25 ergänzt, sondern auch warum. Genau darin liegt der Mehrwert digitaler Lernumgebungen mit LLM-Unterstützung: Erklärungen, Visualisierung und Interaktion können gezielt aufeinander abgestimmt werden.
