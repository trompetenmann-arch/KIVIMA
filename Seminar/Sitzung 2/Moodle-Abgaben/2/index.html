Quadratische Ergänzung mit Flächenmodell

Die Gleichung lautet: x2+10x=39x^2 + 10x = 39x2+10x=39. Die rechte Seite wird als Fläche aus 39 Einheitsquadraten dargestellt.

1. Ausgangssituation

x2+10x=39x^2 + 10x = 39x2+10x=39

Rechts sehen wir 39 Einheitsquadrate. Links wird die Fläche aus x2x^2x2 und 10x10x10x aufgebaut.

2. Zerlegen von 10x10x10x

Die Fläche 10x10x10x wird in zwei gleich große Rechtecke zerlegt: 5x5x5x und 5x5x5x.

x2+5x+5x=39x^2 + 5x + 5x = 39x2+5x+5x=39

3. Zum Quadrat ergänzen

Es fehlt noch ein kleines Quadrat mit Seitenlänge 555, damit links ein großes Quadrat entsteht.

x2+10x+25=39+25x^2 + 10x + 25 = 39 + 25x2+10x+25=39+25

4. Neues Quadrat

Nun ist links ein Quadrat mit Seitenlänge x+5x+5x+5 entstanden.

(x+5)2=64(x+5)^2 = 64(x+5)2=64

5. Seitenlänge bestimmen

Da hier eine Seitenlänge betrachtet wird, nehmen wir die positive Wurzel:

x+5=64=8x + 5 = \sqrt{64} = 8x+5=64​=8

x=3x = 3x=3
