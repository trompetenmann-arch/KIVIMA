# KIVIMA

Projektseite für das KIVIMA-Projekt und das teutolab mathematik.

## Inhalte

- `index.html` – Startseite mit Projektüberblick
- `materialien.html` – Unterseite mit Links auf verfügbare Dokumente
- `seminar.html` – Unterseite zum Seminarablauf
- `mitmachen.html` – Unterseite für Kontakt und Teilnahme
- `styles.css` – gemeinsames Styling der Seite

Die bereitgestellten Projektdokumente liegen im Ordner `Projectinfos/`.
